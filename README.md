# В процессе выполнения рабочих тетрадей 13-17 были решены задачи

## Практическая занятие №13 - Service Worker

Реализован Service Worker для кэширования статических ресурсов. Приложение полностью работает офлайн: страница загружается из кэша, заметки сохраняются в localStorage.

---

## Практическая занятие №14 - Web App Manifest

Создан manifest.json с иконками, названием и цветовой темой. Приложение можно установить на устройство - оно запускается в отдельном окне без элементов браузера.

---

## Практическая занятие №15 - HTTPS + App Shell

Настроен локальный HTTPS-сервер. Реализована архитектура App Shell - каркас приложения кэшируется и отображается мгновенно, контент подгружается динамически.

---

## Практическая занятие №16 - WebSocket + Push

Разработан Node.js-сервер с Socket.io для двусторонней связи. Добавлены push-уведомления через VAPID-ключи - сервер отправляет сообщения даже при закрытом браузере.

---

## Практическая занятие №17 - Детализация Push

Добавлены отложенные напоминания с выбором даты и времени. Сервер хранит активные таймеры и отправляет push-уведомление при наступлении заданного времени. В уведомлении предусмотрена кнопка "Отложить на 5 минут".
